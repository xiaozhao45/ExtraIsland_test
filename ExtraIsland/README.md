## 完成的功能:
流畅时钟:
- 带有切换和闪动动画
- 支持关闭秒数
- 支持秒数小字体显示
- 从准确时间服务获取时间(支持禁用,可直接获取系统时间)
- 整点强调动画
- 更好的配置面板

更高级的倒计日:
- 支持精确到秒
- 带有切换动画
- 前后文案可配置
- 从准确时间服务获取时间(支持禁用,可直接获取系统时间)
